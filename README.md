# Running locally
- Install Node.js - https://nodejs.org/en/download/
- Clone repository `git clone https://github.com/ExC0d3/cloud-to-street-vis-challenge.git`
- `cd cloud-to-street-vis-challenge`
- `npm install`
- `npm start`
- Go to localhost:8000
